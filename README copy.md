# DevShowcase Portfolio

Landing Page de alta performance criada com Next.js 15, Tailwind CSS e Framer Motion.

## Stack Tecnológica

- **Framework:** Next.js 15 (App Router)
- **Estilização:** Tailwind CSS v4
- **Animações:** Framer Motion + Three.js (React Three Fiber)
- **Linguagem:** TypeScript
- **Ícones:** Lucide React

## Como Executar

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Estrutura do Projeto

- `/src/components/main`: Componentes estruturais (Hero, Navbar, Footer)
- `/src/components/sub`: Componentes menores (Cards, Texto animado)
- `/src/app/globals.css`: Configuração do tema e Tailwind
